## Bullet Point Style

Organize all output as scannable bullet points for rapid comprehension.

### Structure

- Lead with the most important point
- Group related items under sub-bullets
  - Use indentation for hierarchy
  - Keep sub-points brief
- One idea per bullet
- Active voice, action-oriented

### Formatting Rules

- **Bold** key terms and actions
- `code` for technical elements
- No paragraphs - bullets only
- Maximum 2 levels of nesting
- Use numbered lists for sequences only
